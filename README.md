## ðŸ“Š Dashboard Design Principles Applied

The dashboard was built applying industry-standard BI design principles:

**Chart Selection Rationale**

| Metric Type | Chart Type Selected | Rationale |
|:---|:---|:---|
| Trend over time | Line / Area chart | Communicates directional change and rate clearly |
| Category comparison | Bar / Column chart | Enables side-by-side magnitude comparison |
| Proportional breakdown | Pie / Donut chart | Highlights share-of-whole relationships |
| KPI summary | KPI tiles / Cards | Delivers instant at-a-glance key metrics |

**Layout & UX Considerations**
- Multi-tab or multi-page layout to separate thematic analytical sections
- Consistent colour palette applied across all panels for visual coherence
- Descriptive chart titles and axis labels to enable self-service interpretation
- Filter widgets for interactive slicing (where applicable)
